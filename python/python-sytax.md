# Frequently used Python Syntax
### print statement
```python
print('hello world)
```

### reverse the array element
```
# reverse all elements
list = list[::-1]

# use reversed() method
reversed(list) 

for x in reversed(whatever):
    do_something(x)
```

### find if the list has the same value
```python
return all(i == list[0] for i in list)   
```
